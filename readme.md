## joiappv2

- Language: Python (3.13.11)
- Framework: FastAPI
- Database: Firebase Firestore
- Authentication: JWT (Access / Refresh Token)
- Dependency Management: requirements.txt (version-pinned)

## How to run Kakao (Android)
- npm install
- npx expo run:android (kakao-expo-test)
- npx react-native run-android (KakaoTestApp)
